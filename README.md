# md-viewer

`md-viewer` 是一款為 macOS 打造的高效能 Markdown 閱讀器，致力於提供極致的閱讀體驗與流暢的編輯預覽整合。

## 主要功能

- **高效能解析引擎**：核心採用 Apple `swift-markdown`，解析速度極快，即使處理大型文件也能保持流暢。
- **即時渲染**：採用非同步渲染架構，確保大型 Markdown 檔案載入與解析時 UI 不卡頓。
- **豐富的主題風格**：內建 6 種閱讀模式，包含 Light, Dark, Sepia, Solarized, Nord 及自動切換模式，滿足各種光線環境。
- **語法高亮**：整合 Highlight.js，自動針對 Markdown 中的程式碼區塊進行語法著色。
- **國際化支援**：完整支援繁中、簡中、英文、日文、韓文，界面與原生選單欄同步切換。
- **檔案系統整合**：
  - 支援 `.md`, `.markdown`, `.txt` 檔案關聯，支援雙擊直接開啟。
  - 智慧連結解析：外部網址跳轉至預設瀏覽器，本地連結提供 App 內載入或 Finder 跳轉功能。

## 快捷鍵指南

| 功能 | 快捷鍵 |
| :--- | :--- |
| 開啟檔案 | `⌘O` |
| 重新載入 | `⌘R` |
| 設定面板 | `⌘,` |
| 放大/縮小 | `⌘+` / `⌘-` |
| 重置縮放 | `⇧⌘R` 或 `⌘0` |
| 全螢幕 | `⌘F` |
| 結束程式 | `⌘Q` |

## 快速開始
1. 將 `md-viewer.app` 移動至 `/Applications`。
2. 雙擊任何 `.md` 檔案，若為首次開啟，請在「取得資訊」中將其設定為預設開啟程式。
3. 按下 `⌘,` 即可進入設定面板自訂您的閱讀體驗。

## 編譯環境需求

   * 編譯環境需求：Swift 6.3+, Go 1.26+, iconutil (macOS 原生)。
   * 核心關鍵路徑：
       * 解析邏輯：core/renderer.go -> CGO -> libMarkdownEngine.dylib。
       * 界面渲染：main.go (WebView) -> 注入的 JS (zoom, theme, i18n)。
       * 打包流程：build.sh (處理 rpath 與 dylib 的關鍵指令)。
